uc_bitcoin by erik78se

Drupal 7 implementation for uc_bitcoin

This is a fork of the uc_bitcoin written by Mike Gogulski. The goal is to create this module widely used in Drupal 7 + Ubercart 2.x

Dependencies

Ubercart 2.x Drupal 7.x PHP requirements: PHP5 cURL support SSL support (if you're using HTTPS to talk to bitcoind)
